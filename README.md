# 🚀 SC26

**SC26** is an AI-powered equity research platform that turns financial data into **clear, actionable investment insights** using legendary investor frameworks.

---

## 🧠 What SC26 Does

SC26 helps you analyze stocks like a professional investor by combining:

* 📊 Structured financial analysis
* 🤖 AI-generated insights
* 📘 Beginner-friendly explanations

It simulates how top investors like Warren Buffett, Philip Fisher, and Peter Lynch would evaluate a company.

---

## ✨ Features

* 🎯 **Legend Investor Lens**

  * Buffett → Value & Moat
  * Fisher → Growth & Quality
  * Lynch → Story & GARP

* 📉 **Risk Analysis**

  * Red flags (fraud, NPAs, governance)
  * Green flags (growth, CASA)

* 📊 **Financial Dashboard**

  * PAT, NPA, Cost-to-Income
  * Business model insights

* 📘 **Beginner Mode**

  * Simple 5-point explanation

* ⚡ **AI Reports**

  * Structured analysis via Claude API

---

## 🏗️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js (Serverless)
* Deployment: Vercel
* AI: Claude API (Anthropic)

---

## 🔐 Security

* API keys stored in environment variables
* No frontend exposure
* Secure backend route `/api/analyze`
